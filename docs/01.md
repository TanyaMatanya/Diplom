npm add --global nx 
добавить в свою систему возможность раоты с репозиторием



nx init
создает новый репозиторий в  этой папке





npm install --save-dev @nx/nest

npm install --save-dev @nx/next

npm install --save-dev @nx/js
Установка библиотек для работы с фреймворками



npx nx g @nx/nest:app --name=api-admin --directory=apps/api/admin --tags=type:api,scope:admin
установка пустого шаблона фреймворка



 api admin нужен для того что бы обращаться с базой даных на уровне администратора



npx nx serve api-public
Запуск елемента моего диплома 


благодаяря эти трем командам я создал три составляшие моего проекта 
доступ для пользователей моего проекта



api это интерфейс к базе данных ответом является только данные сайт эту часть не строит 
admin новые товары о есть часть управления пользователя
public отдача информации простым пользователем
notifications нужен для рассылки сообщений своим пользователям 


npx nx g @nx/next:app  --name=web-admin --directory=apps/clients/admin --tags=type:ui,scope:admin
для админа отдельный вебсайт


npx nx g @nx/next:app  --name=web-public --directory=apps/clients/public  --tags=type:ui,scope:public
для пользователя отдельный вебсайт
